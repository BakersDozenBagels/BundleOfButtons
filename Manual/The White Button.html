On the Subject of The White Button

This is a button. It is white.

The module contains a white button, a ring of cycling red, green, and blue lights, and five blobs of varying colors.

Upon tapping the button, the blob with the lights on will have its color channel adjusted based on the lit up color channel and the seconds digit on the timer.

Pressing the button when the last digit of the timer is even adds the lit up color channel to the lit up blob. Pressing the button when the last digit of the timer is odd removes the lit up color channel to the lit up blob.

Holding the button over a timer tick submits the answer.

Further notes:

  • Pressing the button on x modulo 2 adjusts the y color channel.
    • 0 → Red; 1 → Blue; 2 → Green
  • Pressing the button on x modulo 3 affects the y color channel.
    • 0 → Adds; 1 → Subtracts
  • Pressing the button on x modulo 5 affects the y blob.
    • 0 → Top; 1 → Top Left; 2 → Bottom Left; 3 → Bottom Right; 4 → Top Right
  • The color channels are not as sharp as (e.g. (0, 255, 255)). Each channel is determined by the following formula:
    • 60 + 70n, where n is 0, 1, or 2.
